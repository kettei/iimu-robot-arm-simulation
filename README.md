# imu-robot-arm-simulation
imu-robot-arm-simulation is a matlab code to simulate Robot Arm movements using IMU data (WitMotion Sensor).

## Table of Contents

- [Package](#package)
- [Run](#run)
- [Evidence](#evidence)

## Package
- MATLAB Simulink
- MATLAB Signal Processing Toolbox

## Run

```
open PbD.mlapp > Click Run > See PbD Dashboard > Click Run Program > Click Robot Arm Simulation > Update Waypoints > Click Robot Arm Simulation
```
